# **App Name**: Campus Hub

## Core Features:

- Dashboard Overview: Provides a snapshot of key metrics across all categories (staff, faculty, students, courses, etc.).
- Personnel Management: Modules for managing data related to employees, lecturers, and students.
- Academic Monitoring: Tracks attendance and performance in various class settings (online, in-person, practical exercises).
- Incident Tracking: Record details regarding different reported violations such as violations and other negative behaviors. 
- Statistical Reporting: Generates daily and comprehensive reports based on collected data.
- Access Controls: Allows for different permissions, granting permissions only where necessary to guarantee proper access controls
- Anomaly Detection Tool: Uses generative AI to detect anomalies, such as attendance discrepancies and grade inflation, flagging these anomalies so that managers can take action, providing suggestions to better handle reported incidents based on historical records. The AI tool will provide its reasoning.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), providing a clear and focused feel, communicating intelligence, trust and safety.
- Background color: Light cyan (#E0FFFF), nearly desaturated but still tinted, easy on the eye and calming
- Accent color: Violet (#8A2BE2), attention-grabbing color that creates contrast, drawing attention to actions that can be taken by the user.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective, neutral feel. Note: currently only Google Fonts are supported.
- Simple, modern icons to represent different modules and actions.
- Responsive layout adapts to different screen sizes (desktop, tablet, mobile).
- Subtle transitions and animations to enhance user experience.
